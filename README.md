# Recipe Book

A modern recipe management application built with Angular.

## Features

- Add Recipe
- Edit Recipe
- Delete Recipe
- Favorite Recipes
- Search
- Category Filtering
- Responsive Design
- LocalStorage Support

## Technologies

- Angular
- TypeScript
- SCSS
- RxJS
- Angular Signals
- Reactive Forms

## Installation

```bash
npm install
ng serve
```
