# Recipe Book

A modern recipe management application built with Angular.

## Features

- Add new recipes
- Edit existing recipes
- Delete recipes
- Mark recipes as favorites
- Search recipes
- Filter by category
- Responsive design
- LocalStorage data persistence

## Technologies

- Angular 20
- TypeScript
- SCSS
- RxJS
- Angular Signals
- Reactive Forms
- LocalStorage

## Project Structure

The project follows a feature-based architecture.

```
src/
 ├── core/
 ├── shared/
 ├── features/
 │    └── recipes/
```

## Installation

```bash
npm install
ng serve
```

Open:

```
http://localhost:4200
```

## Technical Features

- Standalone Components
- Lazy Loading
- Reactive Forms
- Signals
- RxJS BehaviorSubject
- Custom Validator
- Custom Pipe
- Custom Directive
- Route Guard
