# Recipe Book

A modern recipe management application built with Angular.

## Live Demo

**Vercel:**  
https://recipe-book-ij8dx7mvk-metehantasci1.vercel.app/recipes

## Project Description

Recipe Book is a modern web application that allows users to create, edit, delete and manage their own recipes. The application is built with Angular and stores all data locally using LocalStorage without requiring a backend.

## Features

- ✅ Add new recipes
- ✅ Edit recipes
- ✅ Delete recipes
- ✅ View recipe details
- ✅ Favorite recipes
- ✅ Search recipes
- ✅ Filter by category
- ✅ Responsive design
- ✅ LocalStorage persistence

## Technologies

- Angular 20

- Angular Signals

## Installation

```bash
git clone https://github.com/metehantasci/recipe-book.git
cd recipe-book
npm install
ng serve
```

Open your browser:

```
http://localhost:4200
```
